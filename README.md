# MIȘU — Asistent Documente Juridice

**Mișu** este un asistent de documente juridice pentru piața românească, bazat pe [Mike](https://github.com/willchen96/mike) de willchen96.

**Tech stack:** Next.js + Cloudflare Pages + Hono Workers + D1 + Drizzle ORM + R2

---

## Caracteristici

- Încărcare și analiză documente juridice (PDF, DOCX)
- Generare rezumate și extrase cu AI
- Asistent AI pentru drafting contracte
- Căutare în legislația românească
- Template-uri documente legale

## Stack Tehnic

| Componentă | Tehnologie |
|---|---|
| Frontend | Next.js 16 pe Cloudflare Pages |
| Backend API | Hono Workers pe Cloudflare Workers |
| Database | Cloudflare D1 (SQLite) |
| ORM | Drizzle ORM |
| Storage | Cloudflare R2 |
| Auth | Custom JWT |
| AI | Claude / Gemini / OpenAI |
